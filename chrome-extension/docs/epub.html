Description: This model is responsible for the read-only attributes and content of an epub. Rationale: This is designed as a model to represent the state of an epub as it is maintained by the Readium application. As Readium does not have any authoring capabilities (a saved epub is not modified), this model essentially represents the epub in a read-only fashion (although this is not enforced).
