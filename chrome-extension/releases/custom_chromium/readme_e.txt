1. Introduction
---------------
This document contains information about Custom Chromium for Readium.
Please read this document carefully before utilizing Custom Chromium for Readium.

-----------
2. Overview
-----------
Name: Custom Chromium for Readium project
Operation verification environment: Mac OS X 10.7.3 and Readium extension Version 0.3.1.

------------------------------
3. How to Use
------------------------------
a. Unzip Chromium-1025-2012-05-17.zip on Mac, Chromium-1025-2012-05-17 will be created.
b. Open Chromium-1025-2012-05-17.
c. Use Chromium just as Chrome browser.

----------------------------------------------------------------
4. How to Use Readium extension with Japanese vertical contents
----------------------------------------------------------------
a. Open http://readium.org/ with Custom Chromium for Readium and click "ADD TO CHROME" button to install Readium extension.
b. Open a new tab, select Apps and click Readium icon to run Readium extension. 
   (http://support.google.com/chrome/bin/answer.py?hl=en&answer=95451)
c. Click Option button, which looks like a gear, from Library view.
d. Uncheck "Paginate all reflowable ePUB content" and click "Save changes" button.
e. Click Add Book button, input or select a content in "Add Book To Readium Library:" dialog and click "Add Book" button.
f. Content will be analyzed and opened automatically.
